GIT - распределенная система контроля версий 
Это система для отслеживания и ведения истории изменения файлов, в вашем проекте.

Репезиторий - хранилище вашего кода и истории его изменений

GitHub - Веб сайт для размешения  Git - репазиториев в совместнойразработке проектов

Команды Git:
 
1. git init -  она создает новый локальный Git репозиторий у вас на ПК, в той директории которой было прописано команда. (команда вводится один раз)

git add -команда для добавления в файлов в систему отслеживания гита, после этой команды изменение начинают отслеживатся гитом.

git commit -m 'commet' - как только достигаем оптимального момента в разработке, то тогда мы сохраняем и коментарируем изменения в локальный репизиторий.

 git remote add -это команда для того чтобы связать наш локальный репозиторий с удаленным репозиторием, то есть лобавляет подключение к репезиторий на гитхабе.
       git remote add <name: origin> <ссылка_на_репо:ssh>
 git push -команда для отправки всех изменений на локальный репезитории в удаленный репазиторий.
           git push origin <branch(main)>
